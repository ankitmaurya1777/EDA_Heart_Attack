# ❤️ Heart Attack Data Analysis

## 📌 Overview

This project focuses on Exploratory Data Analysis (EDA) of a Heart Attack Prediction dataset. The objective is to understand the factors that influence heart disease risk through statistical analysis and visualizations.

The analysis includes data inspection, cleaning checks, feature transformation, univariate analysis, bivariate analysis, and correlation analysis to uncover meaningful patterns in the dataset.

---

## 🎯 Objectives

- Analyze patient health records related to heart disease.
- Identify patterns and relationships among medical attributes.
- Visualize distributions of numerical and categorical features.
- Explore correlations between variables and heart disease outcomes.
- Generate insights that can support predictive modeling.

---

## 📂 Dataset Information

The dataset contains **303 patient records** and **14 features** including:

| Feature | Description |
|----------|-------------|
| age | Age of patient |
| sex | Gender |
| cp | Chest Pain Type |
| trtbps | Resting Blood Pressure |
| chol | Cholesterol Level |
| fbs | Fasting Blood Sugar |
| restecg | Resting ECG Results |
| thalachh | Maximum Heart Rate Achieved |
| exng | Exercise Induced Angina |
| oldpeak | ST Depression |
| slp | Slope of Peak Exercise ST Segment |
| caa | Number of Major Vessels |
| thall | Thalassemia |
| output | Heart Disease Presence (Target Variable) |

## 📈 Key Insights

- Chest pain type shows a strong relationship with heart disease occurrence.
- Maximum heart rate achieved (thalachh) varies significantly across patients.
- Cholesterol and resting blood pressure contain several outliers.
- Certain exercise-related indicators are strongly associated with heart disease.
- Correlation analysis reveals important relationships among cardiovascular features.
- Numerical feature distributions help identify skewness and potential anomalies.
